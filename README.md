# Assignment
# 내배캠 2주차 Dart 과제 모음

간단한 Dart 콘솔 프로그램 3개를 모아둔 프로젝트입니다.

## 1. 점수 등급 출력 (필수 문제 1번)
- `myScore` 값에 따라 A / B / C 등급을 출력.
- 0점 미만, 100점 초과일 경우 `"유효하지 않은 점수입니다."` 를 출력.

## 2. 장바구니 결제 금액 계산 (필수 문제 2번)
- `cart` 리스트에 담긴 상품 이름을 기반으로 총 금액을 계산.
- `pricing` Map을 사용해 상품별 가격을 관리.
- 등록되지 않은 상품(예: 양말, 가방 등)은 경고 메시지를 출력하고 금액 계산에서 제외.
- 총 금액이 20,000원 초과일 경우 10% 할인 금액을 적용하여 최종 결제 금액을 출력.

## 3. 로또 번호 발급 & 당첨 확인 (도전 문제)
- `SplayTreeSet<int>` 를 사용해 1~45 사이 중복 없는 랜덤 로또 번호 6개를 오름차순으로 발급.
- 고정된 당첨 번호 Set과 교집합을 구해 일치하는 개수에 따라 1등~3등 또는 꽝 여부를 출력.
- 마지막에 `myNumber.clear()`로 발급한 로또 번호를 초기화.
